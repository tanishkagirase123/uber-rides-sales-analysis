# uber-rides-sales-analysis
The primary objective of this Power BI project is to develop a comprehensive dashboard that provides insights into Uber/Ola Taxi ride data. We aim to track and analyze various metrics such as total trips, total fares, and trip distribution by location and time. In a data-driven transportation service like Uber, understanding ride patterns and sales performance is critical for improving operational efficiency, customer experience, and strategic decision-making. However, raw ride data alone does not offer actionable insights unless it is properly analyzed and visualized.
This project addresses the challenge of transforming complex Uber ride data into a clear and interactive Sales Dashboard using Power BI. The goal is to analyze and visualize key metrics such as:
Pickup time trends – to identify peak ride hours and optimize driver allocation
Total rides – to monitor daily, weekly, and monthly ride volume
Payment types – to understand customer preferences (e.g., card, cash, wallet)
Revenue distribution – to evaluate income by region or time period
Ride locations – to determine high-demand areas and route efficiency
By building a comprehensive dashboard, this project aims to help stakeholders gain valuable insights into ride behaviors, payment trends, and performance patterns, ultimately supporting data-backed business decisions for growth and service improvement.
